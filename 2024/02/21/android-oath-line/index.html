此文件為說明 Android 開發透過 Line 第三方登入流程.

Step

  1. 前往 https://account.line.biz/login?redirectUri=https%3A%2F%2Fdevelopers.line.biz%2Fconsole%2Fchannel%2F2002090266 使用商用帳號登入。

  1. 建立Providers，並Create a new channel，channel type 選LINE Login。

  2. 需注意步驟 App types，Web app 跟 Mobile app 都打勾

  1. 建立完該channel後，點選進入channel，並將Developing 改為Publish 否則會連不到。

  2. 在Basic settings 分頁裡 OpenID Connect 的 Email address permission 要點選 Apply。

  3. 在 LINE Login 分頁裡，新增 web app 的 Callback URL 與 mobile app Android 的 Package names 與 Package signatures

Package signatures 的憑證取得，打開命令列輸入以下指令，替換掉你的路徑，預設的debug.keystore 密碼為 android。

keytool -keystore C:\Users\huaqu.android\debug.keystore -list -v


SHA1 填入 Package signatures

  1. 在 Android studio 專案裡 res 資料夾中將建立的 app 資訊加入 strings.xml。

  2. 在 build.gradle 中添加 LINE SDK 依賴。

  3. 在 AndroidManifest.xml 添加 Internet 訪問權限
    <uses-permission android:name="android.permission.INTERNET" />

  4. onActivityResult 已棄用，以下為完整程式碼
