Fontmaker use bmfont

2017-03-06
font

將圖片轉為字型檔工具，以利於cocos或unity使用

快速使用

建立圖片資料夾

例 建立 abcFont 並將對應的字元圖片貼上

ps 不要使用中文路徑，包含圖片與執行檔的位置，圖片格式必須為32位元才能匯入

修改設定檔

開啟 bmfont.bmfc 設定檔，並將路徑改為字型資料夾名稱，另存檔名為字型名稱

例 abcFont.bmfc

開啟bmfont.exe

開啟 bmfont.exe 點選Options > Load configuration 點選剛設定的 abcFont.bmfc

有載入圖片成功的字元右下會有小藍點，將需要的字元反白，並點選 Options > Save

configuration as… 進行儲存設定檔。

進行輸出

點選Options > Save bitmap font as… 進行輸出到，統一放到output資料夾中方便管理
