CocosCreator Use Webpack-Vendor

2017-03-03
cocos
cocos

使用於COCOS的第三方插件，由於新版本1.4.0無法掃到第三方插件的全域變數，所以改用Webpack打包。

快速使用

加入第三方插件

如要加入 新的第三方插件 將第三方插件 丟入 libs 並打開index.js 手動鍵入import

打開命令列 進行打包

到資料夾位址打開命令列 輸入 npm run build 會將 所有第三方插件打包自dist資料夾內

於Cocos Creator使用

將dist資料夾裡產出的vendor.js丟到Cocos專案中
