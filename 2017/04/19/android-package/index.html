基本 AdMob 寫法

使用 android 廣告插件 AdMob

需在專案”構建”後”發佈”前在專案底下的build\jsb-default 開啟命令列鍵入

申請 AdMob 帳號 並創建一廣告單元

取得廣告單元編號

修改SDKBOX AdMob 參數

在專案\build\jsb-default\res 底下 sdkbox_config.json 修改id改為自己的

編譯後 完成

注意每次重大修改後 重新”構建” 會導致SDKBOX插件移除 必需重新import 修正 sdkbox_config.json 參數
